# Valorant Server Lag Machine

# Installation
 ### Download python and add to PATH:
  ```
 https://www.python.org/downloads/
 ```
 ### Navigate to installation directory in command prompt:
 ```
 cd Downloads
 ```
 ### Install dependencies:
 ```
 python -m pip install -r requirements.txt
 ```
# Run
 ```
 python main.py
 ```
